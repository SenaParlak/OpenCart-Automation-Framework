# OpenCart Automation Framework

## About

A UI test automation framework developed for the OpenCart demo application using Selenium WebDriver, Java, Cucumber, TestNG, and Maven.

## Technologies

* Java
* Selenium WebDriver
* TestNg
* Cucumber
* Maven
* Gherkin
* Page Object Model (POM)

## Tools
* Intelij IDEA
* Git
* GitHub
